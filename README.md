# RADIX 🔍

AI-powered root cause analysis and quality problem-solving assistant for automotive manufacturing engineers.

## What is RADIX?

RADIX (Latin for "root") is a custom GPT designed to help quality engineers solve manufacturing problems faster and more systematically. It guides users through structured problem-solving methodologies used in automotive and industrial manufacturing.

## What RADIX Can Do

- 🔎 Guide 5-Why root cause analysis
- 📋 Generate 8D problem-solving reports
- 🐟 Build Ishikawa (fishbone) diagrams
- 📝 Draft NCR, CAPA, SCAR, A3, and Lessons Learned reports
- 📚 Reference AIAG, VDA, and ISO standards
- 🏭 Apply automotive quality frameworks (APQP, PPAP, FMEA, SPC)

## Report Types Supported

| Report | Description |
|--------|-------------|
| 5-Why | Root cause drill-down |
| 8D | Eight disciplines problem solving |
| Ishikawa | Fishbone cause & effect diagram |
| NCR | Non-conformance report |
| CAPA | Corrective and preventive action |
| SCAR | Supplier corrective action request |
| A3 | Toyota lean problem solving |
| Lessons Learned | Knowledge capture document |
